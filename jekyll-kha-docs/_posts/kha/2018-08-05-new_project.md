Follow the steps
<h3>
 step -1
</h3>

Create an empty folder and open it with the 
kode studio.

<h3>
 step -2
</h3>
Press the f1 button and type Init kha project
and press enter.Now a new khaproject is created

<h3>
 step -3 running
</h3>

To run the project just press f5 and the project 
will start running


<h3>
 step -4 building
</h3>

to build the kha project press f1 and choose Compile kha Project
and press enter
